📋 下一步：推送到 GitHub
================================================================================

选项 1: 使用自动化脚本
----------------------------------------
cd /opt/ip-track
./push-to-github.sh

脚本会提示你输入 GitHub 用户名，然后自动配置并推送。

选项 2: 手动推送
----------------------------------------
1. 在 GitHub 创建仓库
   - 访问: https://github.com/new
   - Repository name: IP-TRACK
   - Description: IP Address to Switch Port Tracking System
   - 选择 Public 或 Private
   - ❌ 不要勾选 "Initialize this repository with a README"
   - 点击 "Create repository"

2. 配置远程仓库（替换 YOUR_USERNAME）
   cd /opt/ip-track
   git remote add origin https://github.com/YOUR_USERNAME/IP-TRACK.git

3. 推送代码
   git push -u origin main

4. 输入认证信息
   - Username: 你的 GitHub 用户名
   - Password: Personal Access Token（不是密码）

5. 创建 Personal Access Token（如果需要）
   - 访问: https://github.com/settings/tokens
   - 点击 "Generate new token" → "Generate new token (classic)"
   - Note: IP-TRACK deployment
   - 勾选: repo (完整权限)
   - 点击 "Generate token"
   - 复制 token（只显示一次！）
   - 推送时使用 token 作为密码
